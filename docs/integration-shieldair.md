# Integración Motor de Oxígeno con ShieldAir

## Esquema general


[ShieldAir-Urban] [ShieldAir-Mars]
│ │
│ (O₂ del aire) │ (O₂ de CO₂ atmosférico)
▼ ▼
└──────────┬───────────────┘
▼
[Motor de Oxígeno]
│
▼
[Energía mecánica / eléctrica]



## Caso 1: Integración con ShieldAir-Urban

- **Función**: Usar la torre urbana para extraer O₂ puro del smog y alimentar un motor de combustión limpia.
- **Aplicación**: Generación eléctrica en ciudades sin emisiones de NOx.
- **Flujo**: Aire smog → ShieldAir-Urban → O₂ concentrado → Motor de Oxígeno → Electricidad.

## Caso 2: Integración con ShieldAir-Mars

- **Función**: Usar la torre marciana para producir O₂ desde CO₂ atmosférico y alimentar un motor para ascenso o generación.
- **Aplicación**: Propulsión para etapas de ascenso desde Marte, generación de energía para bases.
- **Flujo**: CO₂ atmosférico → ShieldAir-Mars → O₂ + H₂ (electrólisis) → Motor de Oxígeno → Empuje.

## Ventajas de la integración

1. **Sin almacenamiento de oxidante**: El O₂ se produce en tiempo real.
2. **Versatilidad**: El mismo motor funciona con aire terrestre o con CO₂ marciano.
3. **Sin emisiones NOx**: En Tierra, al usar O₂ puro, no se generan óxidos de nitrógeno.
4. **Cero emisiones netas (con hidrógeno)**: En Marte, si el H₂ se produce con energía solar/nuclear, el sistema es limpio.
