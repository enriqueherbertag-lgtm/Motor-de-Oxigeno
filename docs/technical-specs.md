# Especificaciones Técnicas - Motor de Oxígeno

## Flujo de aire

| Etapa | Componente | Parámetro |
|-------|------------|-----------|
| Captura | Entrada de aire | Flujo: 0.5–10 kg/s (escala variable) |
| Filtrado | Filtro de partículas | Eficiencia >99% para PM2.5 |
| Separación O₂ | Membranas de grafeno / zeolitas | Pureza O₂: 95–99% |
| Almacenamiento (opcional) | Tanque buffer | Presión: 10–50 bar |
| Combustión | Cámara cerrada | Temperatura: 1800–2500°C |
| Expansión | Turbina | RPM: 10,000–50,000 (escala) |

## Materiales

| Componente | Material sugerido | Propiedad clave |
|------------|-------------------|-----------------|
| Cámara de combustión | Inconel / Cerámica reforzada | Resistencia térmica >2500°C |
| Membranas de O₂ | Grafeno / Zeolitas | Selectividad O₂/N₂ >20 |
| Turbina | Aleación de titanio / Cerámica | Resistencia a fatiga térmica |
| Carcasa | Acero inoxidable 316L | Resistencia a corrosión |

## Combustibles compatibles

| Combustible | Ventaja | Desafío |
|-------------|---------|---------|
| Hidrógeno (H₂) | Cero emisiones (solo H₂O) | Almacenamiento, producción |
| Metano (CH₄) | Infraestructura existente | Emisiones CO₂ |
| Combustible sintético | Captura de CO₂ neutra | Eficiencia energética |
| Hidrazina (para espacial) | Almacenable | Tóxico |
