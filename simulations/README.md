# Simulaciones - Motor de Oxígeno

Este directorio contiene los archivos de simulación para validar el diseño del Motor de Oxígeno.

## Simulaciones planificadas

| Tipo | Software | Objetivo |
|------|----------|----------|
| CFD | ANSYS Fluent / OpenFOAM | Flujo de aire en el separador de O₂, combustión en cámara |
| MEF | ANSYS Mechanical | Tensiones térmicas en la cámara de combustión |
| Termodinámica | MATLAB / Python | Eficiencia del ciclo, consumo de combustible |
| Integración | Simulink | Control del flujo de O₂ y combustible |

## Estado

Los archivos de simulación se encuentran en desarrollo. Se subirán cuando estén disponibles.
